Aplicacion practica

El metodo de Newton se usa para encontrar valores de una variable que satisfacen ecuaciones no lineales, por ejemplo la tasa interna de retorno de un proyecto o el punto de equilibrio en un modelo de oferta y demanda.

En estos casos la ecuacion se modela como f(x) = 0, donde x representa la cantidad que se desea estimar.
